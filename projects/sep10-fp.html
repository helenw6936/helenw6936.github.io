Freedom Project

Context

The freedom project is a year long project that allows students to pick their own topic to make a website that involves the use of technologies using html, css, boostrap, and a tool of their choice.

Content

For the project, I chose the topic performing arts to showcase the technologies involved in theatre to share that knowledge with others. I created a website with a home page that talks about what performinig arts is and another page linked such as part A that talks about the existing technologies but also a part B that involves two new technology concept that I came up.

Reflection

Some challenges that I faced when working on my freedom project was being able to make the images and the image background responsive in the home page and the other pages where it contained images to sized when the screen size changes. It was diffcult to resize the background image since its not the same as other images because it contained background. However, after watching tutorials on youtube I was able to figure out the code that mades the background image responsive One of the most important takeaways that I have learned from working on my freedom project was that watching tutorial on youtube really helped me with being able to understand or debug my code especially when working on my tool css materialize framework. It helped me understand how to use the code with explanations that allowed me to be able to learn the tool on my own and with far less struggles. If I had more time to work on the freedom project, I would incorporate more of my tool materialize css into my freedom project especially the background so that it doesnt look as plain unlike the home page of my project.
