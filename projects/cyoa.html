Cyoa Project

Context

Choose your own adventure in a tennis match is a cyoa project made with my partner so that we can practice our knowledge on linking together files and organizing them.

Content

For the project, my partner and I are both tennis players on the schools girls varsity team, because of this we decided to create a fun and interesting project based on that. The project contains different files, scenarios, and different decisions based on what the player chooses on the theme of a tennis match.

Reflection

One of the challenges that my partner and I had while working on this project was being able to locate where our file is because of how messy it was since it was all over the place, with that kind of organization, when linking the files some of them did not work. Because of this we decided to delete the files and create all of them in the same folder since we made a mistake of creating more than one folder.

One of my major takeaway because of the challenge that I faced with that being organized especially in files and folders is really important to make it easier to navigate through and make sure that there are no mistakes or not being able to find files and folders when working on it.

For my next steps, we would have created more of a creative background with html if there was more time.
